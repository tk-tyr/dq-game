# 勇者の冒険

ドラゴンクエスト風ブラウザRPG。HTML / CSS / JavaScript のみで動作。

---

## ファイル構成

```
dq-game/
├── index.html   # 画面構造
├── style.css    # スタイル（レスポンシブ対応）
├── data.js      # 静的データ（モンスター・装備・アイテム・EXPテーブル）
├── game.js      # コアエンジン（状態管理・UI・入力処理）
├── battle.js    # バトルシステム
└── field.js     # フィールド・装備・ショップ
```

## ストーリー・エリア

魔王ゾーマを最速で倒すことが目的。3つのエリアを順に攻略する。

| # | エリア | 特徴 |
|---|--------|------|
| 1 | 平原 | スタート地点。町あり（宿屋・道具屋・防具屋）。中ボス：ゴーレム |
| 2 | 暗黒洞窟 | 毒モンスター登場。メタルスライム出現。中ボス：ボストロール |
| 3 | 魔王の城 | 最強クラスの敵。ボス：魔王ゾーマ |

中ボスを倒さないと次のエリアに進めない。各エリア間は自由に行き来できる。

## アイテム・道具屋

最大所持数：各7個。フィールドでも「どうぐを　つかう」から使用可能。

| 名前 | 価格 | 効果 |
|------|------|------|
| やくそう | 10G | HP+35 |
| どくけしそう | 12G | 毒を治す |
| 幸運の粉 | 50G | 所持数×10%の経験値ボーナス。エリアによって追加効果あり |

### 幸運の粉の効果

所持数に応じて獲得経験値にボーナスがかかる（1個につき+10%、7個で+70%）。

**7個所持時の追加効果（エリア別）：**

| エリア | 効果 |
|--------|------|
| 平原 | 経験値ボーナスのみ（裏ボス判定にも使用） |
| 暗黒洞窟 | メタルスライムに確定遭遇 |
| 魔王の城 | メタルキングに確定遭遇 |

確定遭遇時は粉を全消費。

## 毒システム

毒持ちの敵に攻撃されると **1/3 の確率**で毒状態になる（HP横に ☠ 表示）。

- バトル中：敵ターン終了後に最大HPの5%ダメージ
- フィールド：歩くたびに最大HPの3%ダメージ（HP1の場合は行動不能）
- 解除方法：`どくけしそう` を使用 / 宿屋に泊まる

毒を使う敵：スライムベス・ポイズントード・デスバイパー・キングサーペント・魔王ゾーマ・深淵の魔神

## 隠し要素・チート

### 1. じゅもんチート（バトル中）

バトル中に「じゅもん」を **7回連続で選択** する。

- 通常敵：敵のHPが1になる
- ボス・裏ボス：**自分のHP・MPが1になる**（逆効果）

### 2. 宿屋チート（平原）

所持金不足の状態で宿屋への宿泊を **7回連続で失敗** する。

→ 特別イベント発生。**LV99に到達**（HP1025・MP407・ATK310・DEF204・全呪文習得・99999G）

> 通常レベル上げで全て最小値だった場合のステータス。ロト全装備を購入すると深淵の魔神にギリギリ勝てる強さ。

### 3. 暗黒洞窟連続往来チート

平原 ↔ 暗黒洞窟 ↔ 魔王の城 のいずれかのルートで暗黒洞窟を **7回連続で通過** する。

→ **プレイ時間が24時間進む**（繰り返し可能）

> **連続**が条件。途中で「歩く」「宿屋」「道具屋」「防具屋」「装備確認」「どうぐ使用」「ボスへの挑戦」のいずれかを行うとカウントが0にリセットされる。

### 4. 裏ボス出現（平原）

以下の**両方**を満たした状態で平原を歩く：

| 条件 | 内容 |
|------|------|
| アイテム | やくそう・どくけしそう・幸運の粉を各7個所持 |
| 装備 | **ロトの剣・ロトの鎧・ロトの盾をすべて装備中** |

または平原で **777回エンカウント** する。

→ 深淵の魔神が出現（裏ボス）。撃破すると**真のエンディング**。

### 6. 防具屋購入失敗チート

ぼうぐやで **そのカテゴリで売っている最下位の装備** を **所持金不足のまま7回連続で購入しようとする**。

→ その装備を**無料で入手**（下位装備はすべて売り切れになる）。

**例：** 所持金30G・銅の剣が最下位の場合、銅の剣を7回購入失敗 → 銅の剣を入手。鉄の剣は最下位でないため、同じ状況で鉄の剣を7回失敗しても発動しない。

> **ロト装備はこの方法では入手不可。**  
> 通常購入、またはぼうぐやを「やめる」で出るとカウントがリセットされる。

---

### 5. デス！（LV99習得）

LV99で習得。MP450消費。ボス・裏ボスを含む全ての敵に即死が有効。

> 高MPが必要なため、通常プレイでの使用は上級者向け。

---

## 24時間超えモード

プレイ時間（ゲーム内計測）が24時間を超えると発動。

- 全ての敵が戦闘開始時に**哀れみのセリフ**を言い、自分のHPを1にする
- 「たたかう」を選ぶと敵は自動降参する

暗黒洞窟往来チートでプレイ時間を意図的に増やすことで発動可能。
