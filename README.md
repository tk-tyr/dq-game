# 勇者の冒険

ドラゴンクエスト風ブラウザRPG。HTML / CSS / JavaScript のみで動作。

---

## ファイル構成

```
dq-game/
├── index.html   # 画面構造
├── style.css    # スタイル（レスポンシブ対応）
└── game.js      # ゲームロジック全体
```

---

## 動作環境

- PC ブラウザ（Chrome / Firefox / Edge / Safari）
- iPhone / Android（Safari / Chrome）
- オフライン動作可（Google Fonts のみネット接続時に適用）

### スマホでの遊び方

3ファイルを同じフォルダに入れてiPhoneに転送し、「ファイル」アプリで `index.html` をタップ。

## ストーリー・エリア

魔王ゾーマを倒すことが目的。3つのエリアを順に攻略する。

| # | エリア | 特徴 |
|---|--------|------|
| 1 | 平原 | スタート地点。町あり（宿屋・店）。中ボス：ゴーレム |
| 2 | 暗黒洞窟 | 毒モンスター登場。メタルスライム出現。中ボス：ボストロール |
| 3 | 魔王の城 | 最強クラスの敵。ボス：魔王ゾーマ |

中ボスを倒さないと次のエリアに進めない。各エリア間は自由に行き来できる。

## 毒システム

毒持ちの敵に攻撃されると **1/3 の確率**で毒状態になる（HP横に ☠ 表示）。

- バトル中：敵ターン終了後に最大HPの5%ダメージ
- フィールド：歩くたびに最大HPの3%ダメージ（HP1の場合は歩けない）
- 解除方法：`どくけしそう` を使用

毒を使う敵：スライムベス・ポイズントード・デスバイパー・キングサーペント・魔王ゾーマ・深淵の魔神

## 隠し要素・チート

### 1. じゅもんチート（バトル中）

バトル中に「じゅもん」を **7回連続で選択** する。

- 通常敵：敵のHPが1になる
- ボス・裏ボス：**自分のHP・MPが1になる**（逆効果）

### 2. 宿屋チート（平原）

所持金不足の状態で宿屋への宿泊を **7回連続で失敗**する。

→ 特別イベント発生。**ステータスカンスト**（HP/MP/ATK/DEF最大・LV99・全呪文習得）

### 3. 暗黒洞窟連続往来チート

平原 ↔ 暗黒洞窟 ↔ 魔王の城 のいずれかのルートで暗黒洞窟を **7回連続で通過**する。

→ **プレイ時間が24時間進む**（繰り返し可能）

> **連続**が条件。途中で歩く・宿屋・ショップ・装備確認・ボスへの挑戦のいずれかを行うとカウントが0にリセットされる。

### 4. 裏ボス出現（平原）

以下の**両方**を満たした状態で平原を歩く：

| 条件 | 内容 |
|------|------|
| アイテム | やくそう・どくけしそう・幸運の粉を各7個所持 |
| 装備 | ロトの剣・ロトの鎧・ロトの盾をすべて装備中 |

または平原で **777回エンカウント** する。

→ 深淵の魔神が出現（裏ボス）。撃破すると**真のエンディング**。

### 5. デス！（LV99習得）

LV99で習得。MP999消費。ボス・裏ボスを含む全ての敵に即死が有効。

## 24時間超えモード

プレイ時間（ゲーム内計測）が24時間を超えると発動。

- 全ての敵が戦闘開始時に**哀れみのセリフ**を言い、自分のHPを1にする
- 「たたかう」を選ぶと敵は自動降参する

暗黒洞窟往来チートでプレイ時間を意図的に増やすことで発動可能。
